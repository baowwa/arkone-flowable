# 快速开始 - 批量数据录入功能

## 🚀 5分钟快速体验

### 步骤1: 打开原型页面 (30秒)

1. 用浏览器打开 `prototypes/index.html`
2. 在导航页面找到"批量数据录入"卡片
3. 点击进入批量录入页面

### 步骤2: 体验表格批量录入 (2分钟)

1. **智能填充**
   - 点击工具栏的"⚡ 智能填充"按钮
   - 观察所有样本的"操作人"和"操作时间"自动填充

2. **录入数据**
   - 点击第1行的"核酸浓度"单元格
   - 输入: `85.5`
   - 观察"补ddH₂O"自动计算为: `41.18`
   - 按Tab键跳到"OD260/280",输入: `1.85`
   - 按Tab键跳到"OD260/230",输入: `2.15`
   - 按Enter键跳到第2行的"核酸浓度"

3. **查看验证**
   - 在"核酸浓度"输入一个超出范围的值(如: `1500`)
   - 观察单元格显示红色边框
   - 鼠标悬停查看错误提示

4. **查看进度**
   - 观察顶部进度显示从"0/96"变为"1/96"
   - 已完成的行显示蓝色背景

### 步骤3: 体验板位图一键填充 (2分钟)

1. **打开板位图页面**
   - 点击浏览器后退按钮返回导航页
   - 点击"板位图可视化"卡片

2. **查看板位图**
   - 观察96孔板布局(8行×12列)
   - 灰色圆圈表示空闲,蓝色圆圈表示已使用

3. **一键填充**
   - 在右侧"数据录入模式"面板输入板号: `P096-001`
   - 点击"⚡ 一键填充所有孔位"按钮
   - 点击确认对话框的"确定"

4. **观看动画**
   - 观察绿色波纹从A01开始逐个扩散
   - 孔位颜色从灰色变为蓝色
   - 约3秒完成所有孔位填充

5. **查看结果**
   - 点击任意孔位(如: B03)
   - 在右侧查看孔位详细信息

### 步骤4: 体验视图切换 (30秒)

1. 返回批量录入页面
2. 点击顶部"← 切换到单个录入"按钮
3. 观察页面跳转到单个数据录入页面
4. 点击顶部"批量录入模式 →"按钮
5. 观察页面跳转回批量录入页面

## 🎯 核心优势

### 1. 效率提升
- **传统方式**: 96个样本 × 2分钟 = 192分钟(3.2小时)
- **批量录入**: 96个样本 × 20秒 = 32分钟(0.5小时)
- **效率提升**: 90%+

### 2. 操作便捷
- 键盘导航,无需频繁使用鼠标
- 智能填充,减少重复输入
- 一键填充,自动完成批量操作

### 3. 数据准确
- 实时验证,及时发现错误

- 清晰的状态反馈

## 💡 使用技巧

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| Tab | 跳到下一个单元格 |
| Shift+Tab | 跳到上一个单元格 |
| Enter | 跳到下一行相同字段 |
| Esc | 取消编辑 |

### 批量操作

1. **智能填充**: 一键填充操作人和时间
2. **复制行**: 复制当前行数据到其他行
3. **清空行**: 清空当前行所有数据
4. **清空全部**: 清空所有样本数据

### 数据验证

1. **实时验证**: 输入时立即验证
2. **批量验证**: 点击"验证全部"检查所有数据
3. **错误提示**: 红色边框+鼠标悬停提示

### 草稿管理

1. **自动保存**: 输入时自动保存到本地
2. **手动保存**: 点击"保存草稿"按钮
3. **恢复草稿**: 刷新页面后提示恢复
