# 文档索引

本目录包含 ArkOne Flowable 测序流程管理系统的核心归档文档。

## 📚 核心归档文档

### 1. 竞品分析报告.md
- **用途**: 竞品分析
- **内容**: Benchling、LabWare LIMS、BaseSpace等竞品的功能对比和分析
- **适用人群**: 产品经理、架构师

### 2. 产品需求文档.md
- **用途**: 产品需求定义
- **内容**:
  - 功能需求(含批量数据录入)
  - 用户角色和权限
  - 业务流程
  - MVP范围和分阶段实施
  - 验收标准
- **适用人群**: 产品经理、业务分析师、开发团队

### 3. UI设计方案.md
- **用途**: UI设计规范
- **内容**:
  - 页面设计(样本登记、列表、详情、数据录入、板位图、流程配置)
  - 设计规范(色彩、字体、组件)
  - 交互设计
  - 响应式设计
- **适用人群**: UI设计师、前端开发

### 4. 技术设计文档.md
- **用途**: 技术设计和架构
- **内容**:
  - 系统架构设计
  - 数据库设计
  - API接口设计(含批量操作API)
  - 关键技术实现(Aviator公式引擎、AES/GCM加密)
  - 安全设计
  - 部署方案
  - P0决策结论
- **适用人群**: 架构师、后端开发、前端开发
