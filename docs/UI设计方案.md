## 一、设计系统

### 1.1 色彩系统

**主色调**:
- 主色（Primary）: #409EFF - 科技蓝，代表专业和可靠
- 成功（Success）: #67C23A - 绿色，表示完成状态
- 警告（Warning）: #E6A23C - 橙色，表示需要注意
- 危险（Danger）: #F56C6C - 红色，表示错误或异常
- 信息（Info）: #909399 - 灰色，中性信息

**中性色**:
- 文本主色: #303133
- 文本次色: #606266
- 文本辅助色: #909399
- 边框颜色: #DCDFE6
- 背景色: #F5F7FA

### 1.2 字体系统

- 标题字体: 16px/18px/20px/24px, 字重 500-600
- 正文字体: 14px, 字重 400
- 辅助文字: 12px, 字重 400
- 字体族: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial

### 1.3 间距系统

- 超小间距: 4px
- 小间距: 8px
- 中间距: 16px
- 大间距: 24px
- 超大间距: 32px

### 1.4 组件规范

**按钮**:
- 主要按钮: 蓝色背景，白色文字
- 次要按钮: 白色背景，蓝色边框
- 文字按钮: 无背景，蓝色文字
- 尺寸: 小(28px)、中(32px)、大(40px)

**表单**:
- 输入框高度: 32px
- 标签宽度: 100-120px
- 表单项间距: 16px
- 必填标记: 红色星号

**表格**:
- 行高: 48px
- 斑马纹: 奇数行白色，偶数行 #FAFAFA
- 悬停效果: #F5F7FA
- 边框: 1px solid #EBEEF5

## 三、交互设计

### 3.1 核心交互流程

**样本登记流程**:
1. 点击"新建样本" → 打开登记表单
2. 选择检验项目 → 样本类型选项动态更新
3. 选择样本类型 → 显示将使用的流程预览
4. 填写必填字段 → 实时验证
5. 点击"保存" → 生成样本ID，跳转到样本详情

**数据录入流程**:
1. 从样本列表进入详情页
2. 点击当前节点"开始处理"
3. 填写节点数据字段 → 实时验证和计算
4. 点击"提交" → 数据保存，流程流转到下一节点
5. 显示成功提示，更新流程进度条

**流程配置流程**:
1. 打开流程设计器
2. 从工具栏拖拽节点到画布
3. 连接节点，配置条件
4. 点击节点，在右侧配置属性和字段
5. 点击"预览"测试流程
6. 点击"发布"上线流程

### 3.2 快捷操作

- 扫码枪输入: 自动识别样本编号，快速定位
- 键盘快捷键:
    - Ctrl+S: 保存
    - Ctrl+Enter: 提交
    - Esc: 取消
- 批量操作: 勾选多个样本，统一操作
- 快速筛选: 点击状态标签快速筛选

### 4.1 桌面端 (>1200px)

- 简化导航，顶部Tab切换

- 仅支持样本查询和简单数据录入
- 单列布局
- 大按钮，易于触摸
